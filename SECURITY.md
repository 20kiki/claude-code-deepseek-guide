# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please:

1. **Do not** open a public issue
2. Email the maintainer with details of the vulnerability
3. Allow up to 7 days for a response before any public disclosure

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |

## Scope

This project is an installation guide and does not contain executable code. Security concerns are limited to:

- Outdated links or references to unverified sources
- Misleading instructions that could cause security misconfigurations

If you find incorrect or potentially harmful instructions, please report them as a standard bug report via Issues.
